# A1a — OS-Agnostic Engine Implementation Plan (TDD)

> **For agentic workers:** REQUIRED SUB-SKILLS: superpowers:subagent-driven-development + superpowers:test-driven-development. **Iron law: no production code without a failing test first.** This is real (non-throwaway) code — full TDD, no exceptions.

**Goal:** Build the OS-agnostic engine of the predictive-writing app — the `PlatformAdapter` contract and a deterministic suggestion state machine that schedules completions, discards stale results, invalidates correctly, and handles word/full accept — entirely headless and unit-tested, so the macOS adapter (A1b) plugs into a proven core.

**Architecture:** A Cargo **workspace**. `platform` defines the cross-platform contract (validated in `2026-06-03-cross-platform-review.md` §4). `core` is a pure, I/O-free `SuggestionMachine`: it consumes `Event`s and emits `Command`s — no timers, no threads, no FFI inside it, so every behavior (debounce gating, generation tokens, invalidation, accept) is deterministically testable. `context`/`ranker`/`model_client` are the supporting pure/seam crates. A0's proven seams (`BoundsSource`/`FieldSource`/`Completer`) and tested logic (caret ladder, left-context, suggest pipeline, swallow predicate) are promoted here as real, properly-homed crates.

**Tech Stack:** Rust 1.95 workspace, `cargo test`. `llama-cpp-2` 0.1.146 (`metal`) for the real `LocalModel`. No FFI/GUI in this plan (that's A1b).

**Scope:** A1a only. Out of scope (→ A1b): `platform_macos` FFI, Tauri tray, overlay rendering, CGEventTap, the live event pump. Out of scope (→ A2/A3): personalization store, settings GUI, per-app overrides.

**A0 findings folded in:** use a **base** model default (instruct drifts to chatter); the `Completer`/`BoundsSource`/`FieldSource` seams become the trait shape; warm inference is ~38ms so the machine can request synchronously-ish (A1b decides threading); API signatures captured in A0 FINDINGS apply to A1b.

```rust
pub fn cap_words(text: &str, max_words: usize) -> String {
    text.split_whitespace().take(max_words).collect::<Vec<_>>().join(" ")

pub fn next_word(text: &str) -> (String, String) {
    match text.find(char::is_whitespace) {
        None => (text.to_string(), String::new()),
        Some(_) => {
            let trimmed_end = text.trim_start();
            // first word + following single space, remainder starts at next non-space
            let end = trimmed_end.find(char::is_whitespace).unwrap();
            let word = &trimmed_end[..end];
            let rest = trimmed_end[end..].trim_start();
            (format!("{word} "), rest.to_string())
        }
    }
}
pub fn repetition_penalty(candidate: &str, recent: &str) -> f64 {
    if candidate.is_empty() { return 1.0; }
    if recent.split_whitespace().any(|w| w == candidate) { 0.5 } else { 1.0 }
}
```

```rust
//! Deterministic suggestion state machine. Pure: Event in, Commands out.
use platform::{ux_mode, Capabilities, UxMode};
use context::{left_context, trim_prefix};
use ranker::{cap_words, next_word};

#[derive(Clone, Debug, PartialEq)]
pub enum Event {
    Focus { caps: Capabilities },
    TextChanged { value: String, caret: usize, now_ms: u64 },
    CaretMoved { caret: usize },
    Tick { now_ms: u64 },
    CompletionReady { gen: u64, text: String },
    AcceptFull,
    AcceptWord,
    Dismiss,
}

#[derive(Clone, Debug, PartialEq)]
pub enum Command {
    RequestCompletion { gen: u64, prompt: String },
    ShowGhost { text: String },
    UpdateGhost { text: String },
    Hide,
    Insert { text: String },
}

struct Showing { remaining: String, caret: usize }

pub struct SuggestionMachine {
    caps: Capabilities,
    debounce_ms: u64,
    max_words: usize,
    gen: u64,
    value: String,
    caret: usize,
    pending_since: Option<u64>,
    requested_gen: Option<u64>,
    showing: Option<Showing>,
}

impl SuggestionMachine {
    pub fn new(caps: Capabilities, debounce_ms: u64, max_words: usize) -> Self {
        Self { caps, debounce_ms, max_words, gen: 0, value: String::new(), caret: 0,
               pending_since: None, requested_gen: None, showing: None }
    }
    fn enabled(&self) -> bool { matches!(ux_mode(&self.caps), UxMode::Inline | UxMode::Popup) }
    fn hide_if_showing(&mut self, out: &mut Vec<Command>) {
        if self.showing.take().is_some() { out.push(Command::Hide); }
    }

    pub fn on_event(&mut self, ev: Event) -> Vec<Command> {
        let mut out = Vec::new();
        match ev {
            Event::Focus { caps } => {
                self.caps = caps; self.gen += 1;
                self.pending_since = None; self.requested_gen = None;
                self.showing = None;
            }
            Event::TextChanged { value, caret, now_ms } => {
                self.value = value; self.caret = caret;
                self.gen += 1; self.requested_gen = None;
                self.pending_since = if self.enabled() { Some(now_ms) } else { None };
            }
            Event::Tick { now_ms } => {
                if let Some(since) = self.pending_since {
                    if self.enabled() && now_ms.saturating_sub(since) >= self.debounce_ms {
                        let gen = self.gen;
                        let prompt = trim_prefix(&left_context(&self.value, self.caret)).to_string();
                        self.requested_gen = Some(gen);
                        self.pending_since = None;
                        out.push(Command::RequestCompletion { gen, prompt });
                    }
                }
            }
            Event::CompletionReady { gen, text } => {
                if self.requested_gen == Some(gen) && gen == self.gen {
                    let capped = cap_words(&text, self.max_words);
                    if !capped.is_empty() {
                        self.showing = Some(Showing { remaining: capped.clone(), caret: self.caret });
                        out.push(Command::ShowGhost { text: capped });
                    }
                    self.requested_gen = None;
                }
            }
            // filled in Tasks 5–6:
            Event::CaretMoved { .. } | Event::AcceptFull | Event::AcceptWord | Event::Dismiss => {}
        }
        out
    }
}
```

```rust
use std::num::NonZeroU32;
use std::path::Path;
use llama_cpp_2::llama_backend::LlamaBackend;
use llama_cpp_2::model::{AddBos, LlamaModel as LCModel, Special};
use llama_cpp_2::model::params::LlamaModelParams;
use llama_cpp_2::context::params::LlamaContextParams;
use llama_cpp_2::llama_batch::LlamaBatch;
use llama_cpp_2::sampling::LlamaSampler;

pub struct LlamaModel { backend: LlamaBackend, model: LCModel }

impl LlamaModel {
    pub fn load(path: &Path) -> Option<Self> {
        let backend = LlamaBackend::init().ok()?;
        let model = LCModel::load_from_file(&backend, path, &LlamaModelParams::default().with_n_gpu_layers(999)).ok()?;
        Some(Self { backend, model })
    }
}

impl LocalModel for LlamaModel {
    fn complete(&self, prompt: &str, max_tokens: usize) -> String {
        let cp = LlamaContextParams::default().with_n_ctx(Some(NonZeroU32::new(2048).unwrap()));
        let mut ctx = self.model.new_context(&self.backend, cp).unwrap();
        let toks = self.model.str_to_token(prompt, AddBos::Always).unwrap();
        let mut b = LlamaBatch::new(512, 1);
        let last = toks.len() - 1;
        for (i, t) in toks.iter().enumerate() { b.add(*t, i as i32, &[0], i == last).unwrap(); }
        ctx.decode(&mut b).unwrap();
        let mut s = LlamaSampler::greedy();
        let mut out = String::new();
        let mut cur = b.n_tokens();
        for _ in 0..max_tokens {
            let tok = s.sample(&ctx, b.n_tokens() - 1);
            if self.model.is_eog_token(tok) { break; }
            out.push_str(&self.model.token_to_str(tok, Special::Tokenize).unwrap_or_default());
            b.clear(); b.add(tok, cur, &[0], true).unwrap(); cur += 1;
            ctx.decode(&mut b).unwrap();
        }
        out
    }
}
```
