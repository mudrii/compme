# A1a — OS-Agnostic Engine Implementation Plan (TDD)

> **For agentic workers:** REQUIRED SUB-SKILLS: superpowers:subagent-driven-development + superpowers:test-driven-development. **Iron law: no production code without a failing test first.** This is real (non-throwaway) code — full TDD, no exceptions.

**Goal:** Build the OS-agnostic engine of the predictive-writing app — the `PlatformAdapter` contract and a deterministic suggestion state machine that schedules completions, discards stale results, invalidates correctly, and handles word/full accept — entirely headless and unit-tested, so the macOS adapter (A1b) plugs into a proven core.

**Architecture:** A Cargo **workspace**. `platform` defines the cross-platform contract (validated in `2026-06-03-cross-platform-review.md` §4). `core` is a pure, I/O-free `SuggestionMachine`: it consumes `Event`s and emits `Command`s — no timers, no threads, no FFI inside it, so every behavior (debounce gating, generation tokens, invalidation, accept) is deterministically testable. `context`/`ranker`/`model_client` are the supporting pure/seam crates. A0's proven seams (`BoundsSource`/`FieldSource`/`Completer`) and tested logic (caret ladder, left-context, suggest pipeline, swallow predicate) are promoted here as real, properly-homed crates.

**Tech Stack:** Rust 1.95 workspace, `cargo test`. `llama-cpp-2` 0.1.146 (`metal`) for the real `LocalModel`. No FFI/GUI in this plan (that's A1b).

**Scope:** A1a only. Out of scope (→ A1b): `platform_macos` FFI, Tauri tray, overlay rendering, CGEventTap, the live event pump. Out of scope (→ A2/A3): personalization store, settings GUI, per-app overrides.

**A0 findings folded in:** use a **base** model default (instruct drifts to chatter); the `Completer`/`BoundsSource`/`FieldSource` seams become the trait shape; warm inference is ~38ms so the machine can request synchronously-ish (A1b decides threading); API signatures captured in A0 FINDINGS apply to A1b.

pub fn cap_words(text: &str, max_words: usize) -> String {
    text.split_whitespace().take(max_words).collect::<Vec<_>>().join(" ")
}
pub fn next_word(text: &str) -> (String, String) {
    match text.find(char::is_whitespace) {
        None => (text.to_string(), String::new()),
        Some(_) => {
            let trimmed_end = text.trim_start();
            // first word + following single space, remainder starts at next non-space
            let end = trimmed_end.find(char::is_whitespace).unwrap();
            let word = &trimmed_end[..end];
            let rest = trimmed_end[end..].trim_start();
            (format!("{word} "), rest.to_string())
        }
    }
}
pub fn repetition_penalty(candidate: &str, recent: &str) -> f64 {
    if candidate.is_empty() { return 1.0; }
    if recent.split_whitespace().any(|w| w == candidate) { 0.5 } else { 1.0 }
}
