**Goal:** Prove the four hard FFI unknowns of a Rust macOS predictive-writing app — local llama.cpp+Metal inference, AX caret/context read, stall-free key interception, non-activating ghost-text overlay — with **all our own logic unit-tested test-first behind trait seams**, headless integration tests for IO, and a documented manual-acceptance checklist for the irreducibly-visual/permission boundaries.

**Architecture — seam-based so logic is testable without macOS:**
- `src/lib.rs` exposes pure logic + trait seams. Fakes in tests drive the logic; real FFI impls live in probe binaries.
- Seams (these prototype the A1 `PlatformAdapter`): `BoundsSource` (caret rects), `FieldSource` (focused text+caret), `Completer` (model). Real impls: `accessibility-sys`, `llama-cpp-2`.
- **Honesty rule (testing-anti-patterns):** fakes feed *synthetic data shapes* to test OUR ladder/pipeline/predicate logic. A test must assert on resulting behavior, never "the mock was called." Tests that need real macOS (does AX return a rect for TextEdit? does the panel show?) are headless integration tests (where runnable) or the manual-acceptance checklist (Task 9) — never faked into a green.

**Tech Stack:** Rust 1.95 + `cargo test`; `llama-cpp-2` 0.1.146 (`metal`), `encoding_rs` 0.8 for `token_to_piece` decoding, `objc2` 0.6.4, `objc2-app-kit` 0.3.2, `accessibility-sys` 0.2.0, `core-foundation` 0.10.1, `core-graphics` 0.25.0.

```rust
    pub fn resolve_caret(src: &dyn BoundsSource, caret: isize) -> (CaretTier, Option<ScreenRect>) {
        if let Some(r) = src.bounds(caret, 0) {
```

```rust
//! ATDD acceptance: a real GGUF loads on Metal and a warm short completion is < 500ms.
//! Skips (not fails) if the model isn't downloaded, so CI without the model stays green.
use std::num::NonZeroU32;
use std::path::Path;
use std::time::Instant;
use llama_cpp_2::llama_backend::LlamaBackend;
use llama_cpp_2::model::{AddBos, LlamaModel, Special};
use llama_cpp_2::model::params::LlamaModelParams;
use llama_cpp_2::context::params::LlamaContextParams;
use llama_cpp_2::llama_batch::LlamaBatch;
use llama_cpp_2::sampling::LlamaSampler;

const MODEL: &str = "models/qwen2.5-0.5b-instruct-q4_k_m.gguf";

fn warm_complete_ms(prompt: &str, n: usize) -> Option<u128> {
    if !Path::new(MODEL).exists() { return None; }
    let backend = LlamaBackend::init().ok()?;
    let model = LlamaModel::load_from_file(&backend, MODEL, &LlamaModelParams::default().with_n_gpu_layers(999)).ok()?;
    let cp = LlamaContextParams::default().with_n_ctx(Some(NonZeroU32::new(2048).unwrap()));
    let decode = |warm: bool| -> u128 {
        let mut ctx = model.new_context(&backend, cp.clone()).unwrap();
        let toks = model.str_to_token(prompt, AddBos::Always).unwrap();
        let mut b = LlamaBatch::new(512, 1);
        let last = toks.len() - 1;
        for (i, t) in toks.iter().enumerate() { b.add(*t, i as i32, &[0], i == last).unwrap(); }
        let t0 = Instant::now();
        ctx.decode(&mut b).unwrap();
        let mut s = LlamaSampler::greedy();
        let mut cur = b.n_tokens();
        let mut decoder = encoding_rs::UTF_8.new_decoder();
        for _ in 0..n {
            let tok = s.sample(&ctx, b.n_tokens() - 1);
            if model.is_eog_token(tok) { break; }
            let _ = model.token_to_piece(tok, &mut decoder, true, None);
            b.clear(); b.add(tok, cur, &[0], true).unwrap(); cur += 1;
            ctx.decode(&mut b).unwrap();
        }
        let _ = warm;
        t0.elapsed().as_millis()
    };
    let _ = decode(true); // warm-up (Metal shader compile)
    Some(decode(false))
}

#[test]
fn model_loads_and_warm_completion_is_under_500ms() {
    match warm_complete_ms("The quick brown fox", 12) {
        None => eprintln!("SKIP: model not downloaded at {MODEL}"),
        Some(ms) => {
            println!("warm 12-token completion: {ms}ms");
            assert!(ms < 500, "warm completion {ms}ms exceeded 500ms floor");
        }
    }
}
```
