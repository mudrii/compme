# A0 Spike Implementation Plan (TDD/ATDD)

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development + superpowers:test-driven-development. **Iron law: no production code without a failing test first.** Steps use checkbox (`- [ ]`) syntax.

**Supersedes:** the probe-style tasks in `2026-06-03-a0-spike.md`. Same goal, restructured test-first per user directive ("follow TDD/ATDD", maximal mock harnesses).

**Goal:** Prove the four hard FFI unknowns of a Rust macOS predictive-writing app — local llama.cpp+Metal inference, AX caret/context read, stall-free key interception, non-activating ghost-text overlay — with **all our own logic unit-tested test-first behind trait seams**, headless integration tests for IO, and a documented manual-acceptance checklist for the irreducibly-visual/permission boundaries.

**Architecture — seam-based so logic is testable without macOS:**
- `src/lib.rs` exposes pure logic + trait seams. Fakes in tests drive the logic; real FFI impls live in probe binaries.
- Seams (these prototype the A1 `PlatformAdapter`): `BoundsSource` (caret rects), `FieldSource` (focused text+caret), `Completer` (model). Real impls: `accessibility-sys`, `llama-cpp-2`.
- **Honesty rule (testing-anti-patterns):** fakes feed *synthetic data shapes* to test OUR ladder/pipeline/predicate logic. A test must assert on resulting behavior, never "the mock was called." Tests that need real macOS (does AX return a rect for TextEdit? does the panel show?) are headless integration tests (where runnable) or the manual-acceptance checklist (Task 9) — never faked into a green.

**Tech Stack:** Rust 1.95 + `cargo test`; `llama-cpp-2` 0.1.146 (`metal`), `objc2` 0.6.4, `objc2-app-kit` 0.3.2, `accessibility-sys` 0.2.0, `core-foundation` 0.10.1, `core-graphics` 0.25.0.

**Status of P1:** done (commit `62708f4`) as pre-TDD environment exploration — build links Metal, model loads, `n_ctx_train=32768`, 43s build. Task 8 adds its ATDD acceptance test (model-load integration test).

```rust
    /// Seam: source of text-range bounds (real impl = AX; tests use a fake).
    pub trait BoundsSource {
        fn bounds(&self, location: isize, length: isize) -> Option<ScreenRect>;
    }

    #[derive(Debug, PartialEq, Clone, Copy)]
    pub enum CaretTier { Exact, Derived, None }

    /// Native caret-rect ladder: tier1 zero-length at caret; tier3 prev-char right edge.
    pub fn resolve_caret(src: &dyn BoundsSource, caret: isize) -> (CaretTier, Option<ScreenRect>) {
        unimplemented!()
    }
}

#[cfg(test)]
mod caret_tests {
    use super::caret::*;
    use super::geometry::ScreenRect;

    struct Fake { zero: Option<ScreenRect>, prev: Option<ScreenRect> }
    impl BoundsSource for Fake {
        fn bounds(&self, _loc: isize, length: isize) -> Option<ScreenRect> {
            if length == 0 { self.zero } else { self.prev }
        }
    }

    #[test]
    fn exact_when_zero_length_usable() {
        let f = Fake { zero: Some(ScreenRect { x: 10.0, y: 20.0, w: 2.0, h: 18.0 }), prev: None };
        let (tier, r) = resolve_caret(&f, 5);
        assert_eq!(tier, CaretTier::Exact);
        assert_eq!(r.unwrap().x, 10.0);
    }

    #[test]
    fn derived_uses_prev_char_right_edge() {
        let f = Fake { zero: None, prev: Some(ScreenRect { x: 10.0, y: 20.0, w: 8.0, h: 18.0 }) };
        let (tier, r) = resolve_caret(&f, 5);
        assert_eq!(tier, CaretTier::Derived);
        assert_eq!(r.unwrap().x, 18.0); // 10 + 8
    }

    #[test]
    fn falls_to_derived_when_zero_is_container() {
        let f = Fake {
            zero: Some(ScreenRect { x: 0.0, y: 0.0, w: 2500.0, h: 18.0 }), // container -> rejected
            prev: Some(ScreenRect { x: 10.0, y: 20.0, w: 8.0, h: 18.0 }),
        };
        assert_eq!(resolve_caret(&f, 5).0, CaretTier::Derived);
    }

    #[test]
    fn none_when_nothing_usable() {
        let f = Fake { zero: None, prev: None };
        assert_eq!(resolve_caret(&f, 5).0, CaretTier::None);
    }

    #[test]
    fn no_prev_lookup_at_caret_zero() {
        let f = Fake { zero: None, prev: Some(ScreenRect { x: 1.0, y: 1.0, w: 8.0, h: 18.0 }) };
        assert_eq!(resolve_caret(&f, 0).0, CaretTier::None);
    }
}
```

```rust
    pub fn resolve_caret(src: &dyn BoundsSource, caret: isize) -> (CaretTier, Option<ScreenRect>) {
        if let Some(r) = src.bounds(caret, 0) {
            if usable_caret_rect(r.w, r.h) { return (CaretTier::Exact, Some(r)); }
        }
        if caret > 0 {
            if let Some(r) = src.bounds(caret - 1, 1) {
                if usable_caret_rect(r.w, r.h) {
                    return (CaretTier::Derived, Some(ScreenRect { x: r.x + r.w, y: r.y, w: 1.0, h: r.h }));
                }
            }
        }
        (CaretTier::None, None)
    }
```
